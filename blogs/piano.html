This piano is demonstration of we can make sounds using javascript

It plays bhairav raaga. However technically we cant play "raaga" but we can play notes.

So this guy converted bhairav raaga into western style notes. and made them playable via this piano using js.

You can find his article and more raagas converted into notes here: https://mrmr.io/raag/bhairav
